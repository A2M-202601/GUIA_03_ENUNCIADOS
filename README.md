# Guía 3 - Liberar memoria

Ejercicios que piden memoria y el alumno debe liberarla. La función a implementar debe liberar la memoria de lo recibido por parámetro. En el enunciado se explica cómo se pide la memoria en las pruebas, para que el alumno sepa cómo liberarla.


El **objetivo** de esta guía es repasar de manera práctica qué y dónde se debe liberar memoria.
